JIS A 2201:2017 に基づく計算
  • 通気特性式：Q = a・ΔP^(1/n)
  • 両対数の最小二乗法で傾き(1/n)と切片(log a)を求める
  • 9.8Pa時の通気量：Q₉.₈ = a・9.8^(1/n)
  • 総相当隙間面積：αA = 0.627・√ρ・Q₉.₈（cm²）
  • C値：C = αA / S（小数点1桁・四捨五入）
  • n値の妥当性：一般に 1 ≤ n ≤ 2（小さい隙間→1、単純開口→2）
