# 気密計算 C値（JIS A 2201 準拠）

住宅等の気密測定データから **C値（相当隙間面積）** を計算する、スマホ対応のWebアプリです。
測定した「圧力差 ΔP」と「通気量 Q」を入力すると、JIS A 2201:2017 に基づいて自動計算します。

## 使い方

1. 延床面積 S（m²）を入力
2. 測定データ（3〜5点）の ΔP・Q を入力
3. 「計算する」を押す

## 計算する内容

- C値（相当隙間面積）
- 総相当隙間面積 αA
- 隙間特性値 n（妥当性チェック付き）
- 通気率 a
- 決定係数 R²
- 回帰グラフ（両対数）

## 計算式（JIS A 2201:2017）

- 通気特性式：`Q = a・ΔP^(1/n)`（両対数の最小二乗法で a・n を算出）
- 9.8Pa時の通気量：`Q9.8 = a・9.8^(1/n)`
- 総相当隙間面積：`αA = 0.627・√ρ・Q9.8`（cm²、ρ=空気密度・既定1.2）
- C値：`C = αA / S`（cm²/m²）
